# 📌 Project Overview

The **E-Commerce Sales Performance Analysis Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI**.

This dashboard helps business managers analyze sales performance across different platforms, states, products, brands, payment methods, and order status. It provides real-time insights using interactive charts, KPIs, drill-through pages, bookmarks, slicers, and tooltips.

The project demonstrates how raw sales data can be transformed into meaningful business insights for better decision-making.
